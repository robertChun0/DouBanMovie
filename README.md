DouBanMovie
---------------

项目采用了Material Design+MVP+Retrofit+RxJava  

所有API均由[豆瓣API](https://developers.douban.com/wiki/?title=api_v2)提供

|[retrofit](https://github.com/square/retrofit)|网络请求库|
